# 🌌 Brahmand Management System

A club management and financial intelligence platform developed for the Brahmand Astronomy Club.

## Features

### 📅 Annual Planning
- Year-long event calendar
- Priority tracking
- Event completion monitoring

### 🎯 Event Management
- Plan and manage events
- Track participation and outcomes

### 💳 Expense Management
- Record expenses
- Track spending by department
- Budget utilization monitoring

### 🤝 Sponsor Analytics
- Sponsorship tracking
- Funding analysis

### 📊 Performance Analytics
- Event feedback analysis
- Participation trends
- Performance benchmarking

### 📈 Executive Dashboard
- Financial overview
- Club health metrics
- Strategic insights

## Tech Stack
- Python
- Streamlit
- Pandas
- Plotly
